# 🚌 실시간 인천 버스 도착 안내 봇
공공데이터포털의 API를 활용하여 특정 정류장의 버스 도착 정보를 디스코드 채널로 자동 전송하는 봇입니다.

## 주요 기능
- **실시간 데이터 연동:** 인천 버스 도착 정보 API를 통해 1분마다 갱신
- **자동 알림:** 특정 시간대(2:10, 12, 14분) 집중 알림 모드
- **채널 동적 관리:** 버스 위치에 따라 디스코드 채널 이름을 실시간 업데이트 (`channel.edit`)
- **임베드 메시지:** 남은 시간과 정류장 수를 시각적으로 표현

## 기술 스택
- Python, Discord.py
- 공공데이터포털 API (BusArrivalService)
- Cloudtype (Hosting)
